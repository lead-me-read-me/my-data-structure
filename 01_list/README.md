## :one: Overview

* 포인터를 사용하여 여러 개의 노드(Node)를 연결하는 자료구조

* 노드 (Node)
  * 연결 리스트의 기본 구성 요소
  * 구성요소<u>1</u> : 인접한 노드를 가리키는 ***next*** 라는 이름의 포인터
  * 구성요소<u>2</u> : 해당 노드에 저장하는 ***data***
