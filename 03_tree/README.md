## :one: Overview

* 노드(Node)를 나무(tree) 형태로 연결한 자료구조
* 구성요소
  * ***Root Node*** : tree 의 시작 노드로 이 노드를 통해 트리를 탐색한다.
  * ***Leaf Node*** : 자식 노드를 가지지 않는 노드
  * ***Edge*** : 두 노드를 연결하는 선으로 Root 로부터의 간선 수에 따라 Level 을 결정함
